# 吸收曲线与工作曲线数据处理工具

本项目包含三个 Python 脚本，用于处理化学实验中的吸收光谱数据和工作曲线数据，支持数据可视化、拟合分析和结果导出。

---

## 📁 文件说明

| 文件名                            | 功能说明                                       |
| --------------------------------- | ---------------------------------------------- |
| `Absorption_Curve.py`             | 绘制吸收曲线，找出最大吸收波长                 |
| `Working_Curve.py`                | 绘制工作曲线，进行线性拟合并计算未知样品浓度   |
| `Working_Curve_for_2_Machines.py` | 对比两台仪器的工作曲线数据（旧数据 vs 新数据） |

---

## 🚀 快速开始

### 环境要求
1. 确保安装 Python 3.8+
2. 安装依赖包：

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行方式
#### 1. 吸收曲线绘制
```bash
python Absorption_Curve.py
```
- 输入波长和吸光度数据（空格分隔）
- 自动保存图像：`absorption_curve.png`

#### 2. 工作曲线绘制
```bash
python Working_Curve.py
```
- 输入浓度、吸光度标准数据和未知样品吸光度
- 自动保存图像：`working_curve.png`

#### 3. 双机对比工作曲线
```bash
python Working_Curve_for_2_Machines.py
```
- 直接运行，无需交互输入
- 自动保存对比图：`working_curve_comparison.png`

---

## 📊 功能特性

### ✅ Absorption_Curve.py
- 交互式输入波长与吸光度数据
- 自动标记最大吸收波长（λ_max）
- 坐标标注、网格显示、标题注释
- 图像自动保存（PNG格式）

### ✅ Working_Curve.py
- 过原点线性拟合（A = k·c）
- 计算 R² 相关系数
- 自动计算未知样品浓度
- 图形化标注拟合方程与数据点

### ✅ Working_Curve_for_2_Machines.py
- 对比两组工作曲线（旧仪器 vs 新仪器）
- 并列展示两个子图，便于对比
- 输出拟合方程、R² 和浓度结果

---

## 📝 输入格式示例

### 吸收曲线输入
```
波长（空格分隔）：520 540 560 580 600 620
吸光度（空格分隔）：0.055 0.069 0.084 0.057 0.136 0.195
```

### 工作曲线输入
```
浓度（第一项必须为0）：0.000 0.008 0.016 0.024 0.032 0.040
吸光度（第一项必须为0）：0.000 0.031 0.061 0.195 0.391 0.408
未知样品吸光度：0.064
```
