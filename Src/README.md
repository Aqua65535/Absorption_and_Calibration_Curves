# 吸收曲线 / 工作曲线小工具

一个用来画分析化学实验曲线的小软件。 
能画吸收曲线，也能画标准曲线（工作曲线），顺便帮你算未知样浓度。

## 功能

- **吸收曲线**：输入波长和吸光度，自动画出曲线，标出最大吸收波长（λmax）
- **工作曲线**：输入浓度和吸光度，过原点拟合，输入未知样吸光度就能算出浓度，顺便输出 R²

## 怎么用

1. 下载项目后，运行 `main_gui.py`（你也可以自己打包exe用）
2. 点你要用的功能按钮
3. 按格式输入数据（空格分隔）
4. 点击“开始画图”

### 数据输入格式

吸收曲线：
- 波长：`400 420 440 460 480`
- 吸光度：`0.12 0.28 0.45 0.42 0.35`

工作曲线：
- 浓度：`0 0.1 0.2 0.3 0.4`（第一个必须是 0）
- 吸光度：`0 0.15 0.31 0.46 0.60`（和浓度一一对应）
- 未知样吸光度：直接填数字就行

## 依赖

- Python 3.12+
- matplotlib
- numpy
- tkinter（Python自带）
